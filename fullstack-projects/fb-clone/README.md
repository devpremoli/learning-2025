# Run

## backend-nodejs

Environment Variables
* .env in backendnode.js -> MONGO_URL

Commands
* yarn start
* check in console:
  * Backend server is running!
  * Connected to MongoDB

## frontend-react
* .env in frontend-react -> VITE_REACT_APP_PUBLIC_FOLDER=http://localhost:8800/images/
* yarn run dev
